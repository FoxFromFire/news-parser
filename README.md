# Tengri News Parser

## 📋 Описание
Приложение для парсинга новостей с сайта Tengri News с системой ролевого доступа и сохранением данных в БД.

## 🎯 Функционал
- **Авторизация с ролями**
  - Viewer: доступ только к основному источнику
  - Operator: возможность парсить любой URL
- **Парсинг новостей** - извлечение заголовков, текста, дат
- **Фильтрация** - исключение нежелательного контента
- **Сохранение** - хранение в БД

## 🛠 Технологический стек
- Python 3.x
- BeautifulSoup4 - парсинг HTML
- SQLite/PostgreSQL - хранение данных
- Requests - HTTP запросы

- 🚀 Использование
python main.py
